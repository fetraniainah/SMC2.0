Bienvenue dans l'aide de notre logiciel de gestion de produits pour boutique! 
Voici quelques consignes pour vous aider à utiliser notre application.

Connexion :
Pour utiliser le logiciel, vous devez vous connecter en utilisant votre mot de passe.

Tableau de bord :
Une fois connecté, vous accédez à votre tableau de bord où vous trouverez les listes de produits et un sidebar affichant les options suivantes : 
"Ajouter un produit",
 "Liste des produits",
 "Profil",
 "Aide" et "Se déconnecter".

Ajouter un produit :
Si vous souhaitez ajouter un nouveau produit, cliquez sur le bouton "Ajouter un produit". Vous serez dirigé vers une nouvelle page où vous devrez saisir le nom du produit, le prix et la description.
Assurez-vous que le prix est numérique. 
Les champs nom de produit, prix et description sont obligatoires.

Liste des produits :
Si vous souhaitez modifier ou supprimer un produit, cliquez sur le bouton "Liste des produits". Vous serez dirigé vers une nouvelle page où vous pourrez consulter la liste de tous vos produits. 
Pour modifier ou supprimer un produit, double-cliquez sur le produit en question ou cliquez sur son nom. 
Une nouvelle page s'ouvrira sous forme de dialogue pour vous permettre de modifier ou supprimer le produit. 
Assurez-vous de remplir tous les champs obligatoires.

Profil :
Cliquez sur le bouton "Profil" si vous souhaitez modifier votre mot de passe. Pour cela, vous devez saisir votre mot de passe actuel et le nouveau mot de passe souhaité.
 Assurez-vous de bien enregistrer votre nouveau mot de passe car il n'y a pas de moyen de le récupérer si vous l'oubliez.
 Si vous rencontrez des problèmes pour changer votre mot de passe, veuillez contacter l'éditeur du logiciel.

Aide :
Le bouton "Aide" vous permet d'accéder aux instructions d'utilisation du logiciel.
 Si vous avez des questions ou des difficultés à utiliser notre application, n'hésitez pas à nous contacter.

Nous espérons que ces consignes vous aideront à utiliser notre logiciel de gestion de produits pour votre boutique. Merci d'utiliser notre application.
